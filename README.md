# Healthcare Workforce Equity Conference 2025 (HWEC2025)

A responsive landing page for the Healthcare Workforce Equity Conference 2025, designed according to the brand style guide.

## Project Description

This landing page showcases the upcoming Healthcare Workforce Equity Conference scheduled for May 22, 2025, at the Salem Convention Center. The page follows the official brand guidelines and includes sections for:

- Hero banner
- About the conference
- Keynote speakers
- Conference strands/tracks
- Trade show information
- Registration call-to-action

## Features

- Fully responsive design
- Clean, modern aesthetic following the HWEC2025 brand guidelines
- Accessibility-focused development
- Optimized for all device sizes

## Tech Stack

- HTML5
- CSS3 (with CSS variables for theming)
- Responsive design
- Font Awesome icons

## Color Palette

- Primary: #2A365E

- Accent 1: #FF9C3E
- Accent 2: #4F7E77
- Text Primary: #2A365E
- Text Secondary: #4A4A4A
- Highlight: #E9C46A

## File Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styles

## Setup Instructions

1. Clone the repository
2. Open `index.html` in your browser to view the landing page
